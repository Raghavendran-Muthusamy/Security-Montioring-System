

The application has the following features
•	IT person can track access history of all employees in the organization
•	Manager can access his own and access history of his team members
•	Employee can access their own access history

The interface is  a J2EE website. With following facilities
•	Login page – Static Page
•	Access History Page – Dynamic (Depending on the user)
•	Access Device Details – Dynamic page
•	Send Email Page – Static Page (only for IT admins) 
•	Send Alert Page – Static page (only for IT admins)
•	Logout Page – Static Page
